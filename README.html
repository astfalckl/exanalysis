Coexchangeable process modelling for uncertainty quantification in joint climate reconstruction

This is the accompanying package to “Coexchangeable process modelling for uncertainty quantification in joint climate reconstruction”. This .rmd file pulls some results from pre-computed results; to run the analysis from scratch look in run_analysis.R.

Installation

To execute on your local machine you can install exanalysis direct from GitHub:

devtools::install_github("astfalckl/exanalysis")

and load exanalysis .

library(exanalysis)

Data

The package ships with a number of .rds files that contain the data used in the paper. Namely,

  • model_data that contains the simulation outputs of SST,
  • sst_data that contains proxy reconstructions of SST (note there are some added points to the original MARGO dataset).
  • sic_data that contains proxy reconstructions of SST (note there are some added points to the original MARGO dataset).
